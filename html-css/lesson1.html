CSS Selectors

Selectors define to which elements a set of CSS rules apply.

  • Type selectors
    /* This CSS snippet would select all elements of Type input and set their color to red */
input {
color: red
}
  • Class selectors
    /* This CSS snippet would select all elements that have a class name "address-info" and set their color to red */
.address-info {
color: red
}
  • ID selectors
    /* This CSS snippet would select all elements that have an ID "input-email" and set their color to red */
#input-email {
color: red
}
  • Combining selectors
    /* This CSS snippet would select all elements of Type input that also have a class "address-info" and set their color to red */
input.address-info {
color: red
}

For more information about selectors, check the resources section of this class.

The cascade

The cascade is a fundamental feature of CSS. It is an algorithm defining how to combine properties values originating from different sources. It lies at the core of CSS as stressed by its name: Cascading Style Sheets. (MDN)

Pseudo classes

A CSS pseudo-class is a keyword added to selectors that specifies a special state of the element to be selected. For example :hover will apply a style when the user hovers over the element specified by the selector.

Exercise: Check out this page for a list of all available pseudo classes.
